# AI Usage within Positron

Describe the various ways you can use AI inside Positron.

## Answer:

Positron has something called positron assistant that is built into the interface.  This is an AI feature that works like an assistant to help you with debugging, rewriting and programming code, as well as finishing code chunks for you.  Databot is also another AI tool that can be utalized to help with recs and insights for data analysis and visualizations.  On top of this, you can also integrate your own language learning models into the positron interface in order to utalize different ai tools seamlessly with positron.

::: callout-important
**Key Insight**
You can utalize different ai softwares within positron without having to switch back and forth between different interfaces, making it easier to organize your workflow.
:::
